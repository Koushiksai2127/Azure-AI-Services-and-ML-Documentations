# Azure-AI-Services-Documentation
This repository offers guidance on Azure AI Services, featuring straightforward explanations, practical examples, and links to official documentation.

---

**Azure AI Services** is not just a collection of APIs; it is the engine for modernizing enterprise applications. In the era of Copilots and Agents, Azure AI provides the foundational blocks to build systems that can **See, Hear, Speak, Understand, and Reason.**


## The Azure AI Ecosystem

Azure organizes AI capabilities into a hierarchy of abstraction. Understanding this map is crucial for selecting the right tool for the job.


| Layer | Focus | Key Services |
| :----- | :----- | :----- |
| **1. Azure AI Foundry** | Unified Platform | A unified interface to build, evaluate, and deploy generative AI apps. It connects models, data, and safety.). |
| **2. Cognitive Services** | Pre-built Models | Vision, Speech, Language, Translator. |
| **3. Applied AI Services** | Scenario-Specific | Document Intelligence, Video Indexer, Metrics Advisor. |
